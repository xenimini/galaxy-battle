
# Galaxy Battle - 3D Space Game

Космическая 3D сцена с динамическим освещением, планетой с кольцами и поясом астероидов.

## Требования

- Python 3.x или Node.js
- Любой современный браузер (Chrome, Firefox, Edge, Safari)

## Установка и запуск

### Шаг 1. Перейдите в папку проекта

```bash
cd C:\Users\Ксения Адаменко\Downloads\galaxy-battle
```

### Шаг 2. Запустите локальный сервер

**Способ 1 - Python (рекомендуется)**

```bash
python -m http.server 8000
```

Если Python не установлен, скачайте с https://www.python.org/downloads/

**Способ 2 - Node.js**

```bash
npx http-server -p 8000
```

**Способ 3 - Python 3**

```bash
python3 -m http.server 8000
```

### Шаг 3. Откройте игру в браузере

Откройте браузер и введите в адресной строке:

```
http://localhost:8000
```

## Управление

| Действие | Управление |
|----------|------------|
| Вращение камеры | Левая кнопка мыши + перетаскивание |
| Приближение/отдаление | Правая кнопка мыши или колесико |
| Панорамирование | Средняя кнопка мыши |

## Что вы увидите

- Голубая планета с двумя вращающимися кольцами
- 200 астероидов, вращающихся вокруг планеты
- 2500 мерцающих звезд на фоне
- Динамическое освещение (5 источников света)
- Анимированные тени и блики

## Остановка сервера

Нажмите `Ctrl + C` в окне терминала, где запущен сервер.

Откройте файл `src/config/config.js` для изменения:

- Цвета и плотность тумана
- Позицию и угол обзора камеры
- Интенсивность и цвета освещения
- Скорость анимации света

## Возможные проблемы и решения

### Белый экран или ничего не отображается

**Причина:** Вы открыли файл напрямую (`file:///...`), а не через сервер.

**Решение:** Запустите сервер (python -m http.server 8000) и откройте `http://localhost:8000`

### Ошибка "python не распознан"

**Решение:** Установите Python с официального сайта и при установке отметьте "Add Python to PATH"

### Ошибка CORS

**Причина:** Браузер блокирует загрузку модулей из локальной файловой системы.

**Решение:** Всегда используйте локальный сервер для запуска.

### Порт 8000 уже используется

**Решение:** Используйте другой порт:

```bash
python -m http.server 8080
```

Затем откройте `http://localhost:8080`
